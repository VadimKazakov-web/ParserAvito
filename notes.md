<h3 id="command">команды</h2>
* команда для установки selenium:
    > pip3 install selenium

* команда для установки pyinstaller:
    > pip3 install -U pyinstaller
    * *-U - Обновите все указанные пакеты до последней доступной версии.*

* команда для установки requests:
    > pip3 install requests
